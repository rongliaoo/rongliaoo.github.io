🍊139 Word Break

題意

Given a string s and a dictionary of strings wordDict, return true if s can be segmented into a space-separated sequence of one or more dictionary words.

Note that the same word in the dictionary may be reused multiple times in the segmentation.

給定一個字串 s 和一個字串 wordDict 字典，迴傳true 如果 s 可以分割為一個或多個字典單詞的空格分隔序列。
字典中的單字可以重複出現在s中，但是字典中每個單字是獨立的。

Solution - Dynamic programming

思路

透過 dp 陣列來記錄字串 s是否能夠由 wordDict 拆分。

  • 指標i：代表字串s的拆分情況，如果dp[i] == true，則代表字串s[0:i]可以拆分。

  • 指標j：代表子字串的拆分情況，如果dp[j] == true，則代表字串s[0:j]可以拆分。

  • 若存在 j < i 使得：

    • dp[j] == true（s[0:j] 可以拆分）
    • s[j:i] 在 wordDict 中
    • 則 dp[i] = true，表示 s[0:i] 也可以拆分。

換句話說，嘗試所有可能的切割點 j。
如果 dp[j] == true，代表 s[0:j] 可以被拆分。
如果 s[j:i] 在 wordDict 中，則 dp[i] = true。

因此藉由迴圈逐一遍歷字串，若找到拆分點，則代表字串可以由wordDict 中字串組成。

舉例

步驟解析

dp 狀態 索引範圍 s[0:i] 判斷 s[j:i] 是否在 wordDict dp[i] 結果
dp[0] 0 "" 空字串，預設 true ✅ true
dp[1] 1 "l" "l" 不在 wordDict ❌ false
dp[2] 2 "le" "le" 不在 wordDict ❌ false
dp[3] 3 "lee" "lee" 不在 wordDict ❌ false
dp[4] 4 "leet" "leet" 在 wordDict ✅ true
dp[5] 5 "leetc" "c" 不在 wordDict ❌ false
dp[6] 6 "leetco" "co" 不在 wordDict ❌ false
dp[7] 7 "leetcod" "cod" 不在 wordDict ❌ false
dp[8] 8 "leetcode" "code" 在 wordDict，且 dp[4] = true ✅ true

Code

時間複雜度：$O(n^2)$
