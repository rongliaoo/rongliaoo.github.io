1025. Divisor Game

題意

Alice 和Bob輪流玩遊戲，Alice 先開始。
最初，黑板上有一個數字 n。在每個玩家的回合中，該玩家進行一次移動，包括：

  • 選擇具有 0 < x < n 且 n % x == 0 的任何 x。
  • 將黑板上的數位 n 替換為 n - x。

此外，如果玩家無法移動，他們就會輸掉遊戲。
當且僅當 Alice 贏得遊戲時，假設兩個玩家都以最佳方式玩，則返回 true。

測資

Constraints:

1 <= n <= 1000

Solution

思路

( n ) Alice 可選擇的 ( x ) 下一步 ( n - x ) Alice 是否能贏
1 無 無法移動 ❌ False
2 1 1 (Bob 輸) ✅ True
3 1 2 (Bob 贏) ❌ False
4 1, 2 3 (Bob 輸) ✅ True
5 1 4 (Bob 贏) ❌ False
6 1, 2, 3 5 (Bob 輸) ✅ True
  • 基礎情況
    • 如果n=1，則Alice無法進行動作，輸出為false
    • 如果n=2，則Alice可選擇x=1，則Bob無法動作，輸出為true
  • 考慮偶數與奇數之間的關係
    • 偶數 - 1 -> 奇數
    • 奇數 - 奇數 -> 偶數

因此

  • 如果n為偶數，則Alice只要選擇1，則會變為奇數狀況，

  • 如果n為奇數，奇數的因數只會是奇數，Alice只要選擇其中一個因數作為x，到Bob時必為奇數。

  • 考慮n=3情況，從Base case來看：

    • n=1 -> Alice會輸，
    • n=2 -> Alice會贏，
    • n=3時為奇數，唯一可作為x的因數為1，因此將變為3-1=2，換到Bob回合，可知情況n=2為贏，因此Bob會贏，Alice會輸。
    • 再考慮n=4情況，當4-1之後換到Bob回合，並且轉為奇數情況，可知n=3結果為輸，因此Bob會輸，Alice會贏。

由以上情況可得，如果n為偶數，則會造成贏場面，如果n為奇數，則造成輸場面。

因此只需要判斷給定n為偶數還是奇數即可。
