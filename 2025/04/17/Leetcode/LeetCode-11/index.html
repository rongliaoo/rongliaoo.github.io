🟠11. Container With Most Water

題意

You are given an integer array height of length n. There are n vertical lines drawn such that the two endpoints of the ith line are (i, 0) and (i, height[i]).

Find two lines that together with the x-axis form a container, such that the container contains the most water.

Return the maximum amount of water a container can store.

Notice that you may not slant the container.

給定一個長度為 n 的整數陣列 height。有 n 條垂直線被繪製出來，其中第 i 條線的兩個端點分別為 (i, 0) 和 (i, height[i])。

請找出兩條線，使得它們與 x 軸一起形成一個容器，這個容器能裝下最多的水。

請回傳這個容器最多能儲存多少水。

注意：你不能傾斜容器。
![[Pasted image 20250417115152.png]]

Constraints

  • n == height.length
  • 2 <= n <= 105
  • 0 <= height[i] <= 104

Solution

思路

  • 利用two pointers，從陣列的頭尾開始（left-> index 0, right-> index n-1）。
    • pointer的移動->思考如何獲得最大面積：獲得較高的邊與距離最遠的兩條邊，因此每次移動最小邊的一邊，若相同長度移動left or right。
  • 每一步計算面積的方式：
    • 高h = 兩邊垂直線最小的一邊
    • 寬w = 兩邊垂直線x軸位置相減
    • 最大面積max_area = (right - left) * min(height[left], height[right])

Time complexity: O(N)

Space complexity: O(1)

two pointer 避免了暴力兩層迴圈，列出所有 (i,j)(i, j)(i,j) 配對，但這會是 O(n²)，在 $n≤105n \leq 10^5n≤105$ 下會超時。
