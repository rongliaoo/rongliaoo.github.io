🍀Minimum Operations to Make Array Values Equal to K

題意

You are given an integer array nums and an integer k.

An integer h is called valid if all values in the array that are strictly greater than h are identical.

For example, if nums = [10, 8, 10, 8], a valid integer is h = 9 because all nums[i] > 9 are equal to 10, but 5 is not a valid integer.

You are allowed to perform the following operation on nums:

  • Select an integer h that is valid for the current values in nums.
  • For each index i where nums[i] > h, set nums[i] to h.

Return the minimum number of operations required to make every element in nums equal to k. If it is impossible to make all elements equal to k, return -1.

你被給定一個整數陣列 nums 和一個整數 k。
你可以進行以下操作：

  1. 選擇一個整數 h（稱為 valid），其條件是：所有比 h 大的數都要相同。

  2. 然後將所有 nums[i] > h 的元素設為 h。

目標是透過最少次數的操作，使得所有陣列元素都等於 k。
如果無法做到，則回傳 -1。

Constraints

Solution

Approach 3 → Using Set

思路

  • 如果陣列中出現比 k 小的數，則無法透過「只降低值」的操作達成 → 回傳 -1。

  • 使用 unordered_set 快速去重，統計有多少個不同的元素。

  • 如果最小值就是 k，則最後一次操作不需要進行（因為 k 已經在裡面），所以答案為 set.size() - 1。

  • 否則，所有不同元素都必須做一次操作 → 回傳 set.size()。

時間與空間複雜度

  • Time Complexity: O(n)

  • Space Complexity: O(n)

Approach 4 → Using Bitset

思路

  • 核心：只處理 大於 k 的不同數字，因為這些才需要壓到 k。

  • 每個比 k 大的獨特數字，都要進行一次操作，直到最終達到 k。

  • 使用 bitset 節省空間與加快速度，因為題目中的數字最大為 (10^6)。
