⭐10056 - What is the Probability ?

題目敘述

  • 有 N 名玩家，依照順序輪流擲骰子（或類似的機制）。
  • 每個玩家擲骰子時，以機率 p 獲勝（即發生「成功事件」）。
  • 若某位玩家擲中成功事件，遊戲立即結束，該玩家獲勝。
  • 若失敗，則輪到下一位玩家進行擲骰，依此類推。
  • 若所有玩家一輪都沒中，則遊戲回到第 1 位玩家，繼續下一輪。

Input

  • $S \le 1000$
  • $N \le 1000$
  • $0 < p < 1$
  • $1 \le I \le N$

Output

每組測試數據，輸出第I位玩家的獲勝機率，並四捨五入到小數點後四位。

Solution

先思考第一輪：

  • 第一位玩家（i=1）在第一回合勝利的機率 :
    • $ (1-p)^{i-1}*p$ = $(1-p)^0 *p $
  • 第一回合沒有人勝利，且有兩位玩家：
    • 第一回合皆輸機率且第二回合第一位玩家勝利機率：
      • ${(1-p)(1-p)}{p*(1-p)^0}=(1-p)^2*p$
    • 假設第二回合也無人勝利，則需再加上一回合中兩人皆輸的機率：
      • $(1-p)(1-p){p(1-p)^2}=(1-p)^4*p$

歸納：

  • 第一位玩家勝利的情況 = 第一回合贏 或 第二回合才贏 或 第三回合贏…
  • 公式：
    • $(1-p)^0 p$ $+ (1-p)^2p + (1-p)^4*p$ $+ \dots$
  • 根據規律觀察，可得出通式：
    • $(1-p)^{i-1} *p$ $+ \dots$
    • $(1-p)^{i-1} *p$ $+ (1-p)^{i-1+N}*p + (1-p)^{i-1+2N}*p$ $+ \dots$
    • N為玩家人數，有代表第幾回合
    • 無限等比級數：首項$a_1 = (1-p)^{i-1} *p$，$r = (1-p)^{N}$。
    • 公式：$S_n = \frac{a_1}{1-r} = \frac{(1-p)^{i-1}*p}{1-(1-p)^N}$=。
